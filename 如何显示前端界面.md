# 如何显示完整前端界面

## 🎯 问题说明

你当前访问的是 **后端API地址**：
```
https://my-space-beryl.vercel.app
```

这个地址只返回JSON数据，不是用户界面。

## ✅ 解决方案：部署前端

前端需要单独部署。有两种方法：

---

## 方法1：通过Vercel Dashboard部署（最简单）

### 步骤1：访问Vercel
打开浏览器，访问：https://vercel.com/dashboard

### 步骤2：创建新项目
1. 点击右上角 "Add New..." 按钮
2. 选择 "Project"

### 步骤3：导入仓库
1. 在 "Import Git Repository" 部分
2. 找到 `My_Space` 仓库
3. 点击 "Import" 按钮

### 步骤4：配置项目
**重要配置：**

| 配置项 | 值 |
|--------|-----|
| Project Name | `my-space-frontend` （或任意名称） |
| Framework Preset | Vite |
| Root Directory | 点击 "Edit"，选择 `frontend` ⚠️ |
| Build Command | `npm run build` |
| Output Directory | `dist` |
| Install Command | `npm install` |

**⚠️ 最重要的是设置 Root Directory 为 `frontend`！**

### 步骤5：部署
1. 点击 "Deploy" 按钮
2. 等待1-2分钟
3. 部署成功后会显示前端URL

### 步骤6：访问前端
部署完成后，Vercel会给你一个URL，例如：
```
https://my-space-frontend.vercel.app
```

访问这个URL就能看到完整的前端界面了！

---

## 方法2：通过命令行部署

### 步骤1：安装Vercel CLI
```bash
npm install -g vercel
```

### 步骤2：进入前端目录
```bash
cd frontend
```

### 步骤3：登录Vercel
```bash
vercel login
```

### 步骤4：部署
```bash
vercel --prod
```

### 步骤5：按提示操作
- Set up and deploy? → **Yes**
- Which scope? → 选择你的账号
- Link to existing project? → **No**
- What's your project's name? → **my-space-frontend**
- In which directory is your code located? → **./**
- Want to override the settings? → **No**

### 步骤6：等待完成
CLI会显示部署URL，访问即可。
