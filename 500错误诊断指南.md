## 💡 可能的错误原因

基于经验，500错误最常见的原因：

### 原因1：依赖导入失败（最可能）

```python
from services.pdf_parser import PDFParser  # 可能失败
from services.ai_service import AIService  # 可能失败
from utils.cache import CacheManager       # 可能失败
```

**症状：**
```
ImportError: No module named 'pdfplumber'
或
ModuleNotFoundError: No module named 'openai'
```

**解决方案：**
- 检查 `api/requirements.txt` 是否正确
- 确认所有依赖都已安装

### 原因2：OpenAI API配置问题

```python
OPENAI_API_KEY = os.getenv("OPENAI_API_KEY", "...")
```

**症状：**
```
ValueError: 请配置 OPENAI_API_KEY 环境变量
或
openai.error.AuthenticationError: Invalid API key
```

**解决方案：**
1. 访问 Vercel Dashboard
2. Settings → Environment Variables
3. 确认 `OPENAI_API_KEY` 存在且正确

### 原因3：PDF解析失败

```python
text_content = pdf_parser.extract_text(pdf_content)
```

**症状：**
```
PDFSyntaxError: PDF文件格式错误
或
UnicodeDecodeError: 字符编码错误
```

**解决方案：**
- 使用文本型PDF（可复制文字的）
- 不要使用扫描件或加密PDF

### 原因4：内存或超时限制

Vercel Serverless Functions有限制：
- 内存：1024MB
- 超时：10秒（Hobby计划）

**症状：**
```
Function execution timed out
或
Memory limit exceeded
```

**解决方案：**
- 压缩PDF文件
- 优化代码性能
