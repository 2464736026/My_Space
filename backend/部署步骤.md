# 阿里云部署步骤

## 问题：系统有多个 Python 版本

如果你安装了 Python 3.13 和 Python 3.10，系统可能默认使用 3.13。

### 解决方案

有三种方法：

---

## 方法1：自动检测（推荐）

双击运行：
```
1_install_deps.bat
```

脚本会自动查找 Python 3.10 或 3.9 并使用。

---

## 方法2：手动指定路径

### 步骤1：查找 Python 3.10 路径

双击运行：
```
find_python.bat
```

这会显示所有已安装的 Python 版本和路径。

### 步骤2：使用手动脚本

双击运行：
```
1_install_deps_manual.bat
```

然后输入 Python 3.10 的完整路径，例如：
```
C:\Python310\python.exe
```
或
```
C:\Users\你的用户名\AppData\Local\Programs\Python\Python310\python.exe
```

---

## 方法3：使用 py launcher

在命令行中运行：
```cmd
cd E:\newP\backend
py -3.10 -m pip install -r requirements.txt -t python
```

---

## 第二步：打包代码

依赖安装成功后，双击运行：
```
2_package.bat
```

## 第三步：上传到阿里云

1. 访问：https://fcnext.console.aliyun.com/cn-hangzhou/functions
2. 选择函数 `api`
3. 点击"代码"标签
4. 上传 `resume-analyzer.zip`
5. 点击"部署"

## 第四步：测试

访问：`https://api-lanxianlei.cn-hangzhou.fcapp.run/`

应该看到：
```json
{
  "message": "AI Resume Analyzer API",
  "version": "1.0.0"
}
```

---

## 常见问题

### Q1: 如何确认使用的是哪个 Python 版本？

运行：
```cmd
python --version
py -3.10 --version
```

### Q2: Python 3.10 安装在哪里？

常见位置：
- `C:\Python310\`
- `C:\Users\你的用户名\AppData\Local\Programs\Python\Python310\`
- `C:\Program Files\Python310\`

### Q3: 可以使用 Python 3.9 吗？

可以！阿里云 FC 支持 Python 3.9 和 3.10。

---

## 文件说明

- `find_python.bat` - 查找已安装的 Python 版本
- `1_install_deps.bat` - 自动检测并安装依赖
- `1_install_deps_manual.bat` - 手动指定 Python 路径
- `2_package.bat` - 打包代码和依赖
- `resume-analyzer.zip` - 最终的部署包
