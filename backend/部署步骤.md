# 阿里云部署步骤

## 第一步：安装依赖

双击运行：
```
1_install_deps.bat
```

这会将所有依赖安装到 `python` 目录。

**注意**：
- 需要 5-10 分钟
- 需要网络连接
- 确保 Python 版本是 3.9 或 3.10

## 第二步：打包代码

双击运行：
```
2_package.bat
```

这会生成 `resume-analyzer.zip` 文件。

## 第三步：上传到阿里云

1. 访问：https://fcnext.console.aliyun.com/cn-hangzhou/functions
2. 选择函数 `api`
3. 点击"代码"标签
4. 上传 `resume-analyzer.zip`
5. 点击"部署"

## 第四步：测试

访问：`https://api-lanxianlei.cn-hangzhou.fcapp.run/`

应该看到：
```json
{
  "message": "AI Resume Analyzer API",
  "version": "1.0.0"
}
```

---

## 如果安装依赖失败

### 方案1：升级 pip
```cmd
python -m pip install --upgrade pip
```

### 方案2：使用国内镜像
编辑 `1_install_deps.bat`，在 pip install 命令后添加：
```
-i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 方案3：检查 Python 版本
```cmd
python --version
```
确保是 3.9 或 3.10

---

## 文件说明

- `1_install_deps.bat` - 安装依赖到 python 目录
- `2_package.bat` - 打包代码和依赖
- `resume-analyzer.zip` - 最终的部署包
- `python/` - 依赖库目录（会自动生成）
