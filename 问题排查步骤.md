## 🎯 可能的问题和解决方案

### 问题1：PDF文件是扫描件（图片）

**症状：**
```
PDF文件内容为空或无法解析
```

**解决方案：**
- 使用文本型PDF（可以复制文字的PDF）
- 不要使用扫描件或图片转换的PDF

**测试方法：**
1. 在PDF阅读器中打开文件
2. 尝试选择和复制文字
3. 如果无法复制，说明是扫描件

### 问题2：OpenAI API调用失败

**症状：**
```
AI简历分析失败: ...
```

**可能原因：**
- API Key无效
- API请求超时
- API返回格式错误

**解决方案：**
1. 访问 `https://my-space-beryl.vercel.app/api/test`
2. 检查 `OPENAI_API_KEY` 是否存在
3. 如果不存在，需要在Vercel重新配置环境变量

### 问题3：文件过大

**症状：**
```
413 Payload Too Large
```

**解决方案：**
- 压缩PDF文件
- 使用小于4MB的文件

### 问题4：网络超时

**症状：**
```
Network Error / Timeout
```

**解决方案：**
- 检查网络连接
- 确认VPN是否影响访问
- 重试上传

### 问题5：CORS跨域问题

**症状：**
```
Access to XMLHttpRequest ... has been blocked by CORS policy
```

**解决方案：**
后端已配置CORS，如果仍有问题：
1. 清除浏览器缓存
2. 使用无痕模式测试
3. 检查后端部署状态
