# 🌐 网页部署快速指南

## 5分钟完成前端部署

忘掉命令行！通过网页部署更简单。

---

## 🎯 立即开始

### 第1步：打开Vercel

**访问：** https://vercel.com/dashboard

确认已登录（右上角显示你的头像）

---

### 第2步：创建项目

1. 点击 **"Add New..."** 按钮（右上角）
2. 选择 **"Project"**

---

### 第3步：导入仓库

1. 找到 **`My_Space`** 仓库
2. 点击 **"Import"** 按钮

**看不到仓库？**
- 点击 "Adjust GitHub App Permissions"
- 授权后刷新页面

---

### 第4步：配置项目

#### 必填项：

1. **Project Name**
   ```
   my-space-frontend
   ```

2. **Framework Preset**
   ```
   选择: Vite
   ```

3. **Root Directory** ⚠️ 最重要！
   ```
   点击 "Edit" → 选择 "frontend" 文件夹 → 确认
   ```

#### 自动检测项（通常不需要改）：

- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

---

### 第5步：部署

点击底部的 **"Deploy"** 按钮

等待2-3分钟...

---

### 第6步：访问网站

部署完成后，点击显示的URL：

```
https://my-space-frontend-xxx.vercel.app
```

---

## ✅ 成功标志

看到这些就成功了：

- ✅ 状态显示 "Ready" 🟢
- ✅ 有可访问的URL
- ✅ 网站显示 "AI智能简历分析系统"

---

## ⚠️ 最常见的错误

### 错误：构建失败 "No package.json found"

**原因：** Root Directory 没有正确设置

**解决：**
1. 进入项目 Settings
2. 找到 Root Directory
3. 点击 Edit
4. **必须通过文件选择器选择 `frontend` 文件夹**
5. 不要手动输入！
6. 保存后点击 "Redeploy"

---

## 🔧 重新部署

如果需要重新部署：

1. 进入项目页面
2. 点击最新的部署
3. 点击 "..." 菜单
4. 选择 "Redeploy"

---

## 📊 配置检查

部署前确认这些配置：

```
✓ Project Name: my-space-frontend
✓ Framework: Vite
✓ Root Directory: frontend (通过文件选择器选择)
✓ Build Command: npm run build
✓ Output Directory: dist
```
